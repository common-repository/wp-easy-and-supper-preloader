This plugin is developed to add supper preloaders on your wordpress site.

== Description ==

This is very easy and supper wordpress preloader plugin. You can choose preloader from 14 CSS3 preloader styles and match preloader style with your site color scheme by selecting any color from 10 supper colors. Its really easy to implement just install wordpress plugin and active. Then you can change your preloader style and color. You can select different preloader style for internet explorer. Its also show preloader when a ajax request is running.

= Features =

1. 14 preloader style
2. Change color
3. Very lightweight
4. All are CSS3 preloader so load faster then image preloader
5. Powered by pace.js
6. Cross Broser Compatible

== Installation ==

Installing this plugin is pretty easy like all other wordpress plugins. Here are 2 ways to install :

= Manual Install =
 
Download the package from your download page and unzip it, inside the package you find a plugin folder. Open it and upload the zip through the wordpress plugin importer or upload it throughout your favorite FTP application inside wp-content/plugins/

Find the WP easy and supper Preloader Plugin in your plugin list after you've uploaded it and activate it. Now you see a new menu item "WP Preloader" in the admin menu.

= Automatic Install =

1. Go to Plugins-> Add New from your wordpress Dashboard
2. Search WP easy and supper Preloader 
3. From the results click on 'install now'
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go To 'WP easy and supper Preloader' menu to configure options
